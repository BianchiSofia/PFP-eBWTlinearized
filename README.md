# PFP-eBWT
Implementation of the eBWT using Prefix free parsing (PFP)

# Download

```console
git clone https://github.com/davidecenzato/PFP-eBWT.git
cd PFP-eBWT
git clone https://github.com/simongog/sdsl-lite.git
```

# Compile

```console
make install_sdsl
make 
```
# Usage

```console
bebwt yeast.fasta
```
